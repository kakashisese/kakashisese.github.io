「网络流 24 题」搭配飞行员

问题描述

飞行大队有若干个来自各地的驾驶员，专门驾驶一种型号的飞机，这种飞机每架有两个驾驶员，需一个正驾驶员和一个副驾驶员。由于种种原因，例如相互配合的问题，有些驾驶员不能在同一架飞机上飞行，问如何搭配驾驶员才能使出航的飞机最多。
因为驾驶工作分工严格，两个正驾驶员或两个副驾驶员都不能同机飞行。

输入格式

第一行，两个整数 n与 m，表示共有 n个飞行员，其中有 m名飞行员是正驾驶员。
下面有若干行，每行有 2个数字 a、b。表示正驾驶员 a和副驾驶员 b可以同机飞行。
注：正驾驶员的编号在前，即正驾驶员的编号小于副驾驶员的编号。

输出格式

仅一行一个整数，表示最大起飞的飞机数。

样例输入

10 5
1 7
2 6
2 10
3 7
4 8
5 9

样例输出

4

一点小想法

这是一道二分图匹配问题。
建图思路就是 给可以一起驾驶飞机的正驾驶员和副驾驶员连一条边，即：给每次输入的a和b连边。 连完后跑一遍二分图匈牙利算法求出最大匹配即可。

代码

#include <cstdio>
#include <algorithm>
#include <vector>
using namespace std;
const int Max = 100 + 5;
vector<int> g[Max];
int n, m, vis[Max], lin[Max], I;
int xyl(int x) { //匈牙利算法
    for (int i = 0; i < g[x].size(); i++) {
        int v = g[x][i];
        if(vis[v] != I) {
            vis[v] = I;
            if(!lin[v] || xyl(lin[v])) {
                lin[v] = x;
                return 1;
            }
        }
    }
    return 0;
}

int main() {
    scanf("%d %d", &n, &m);
    int a, b;
    while(scanf("%d %d", &a, &b) == 2) { //读入
        g[a].push_back(b); //连边
    }
    int ans = 0;
    for (I = 1; I <= m; I++) //求最大匹配
        ans += xyl(I);
    printf("%d", ans);
    return 0;
}
